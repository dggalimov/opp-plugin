# Правила проекта

> Правила этого проекта. По умолчанию заполнены; правьте под себя — в окне (раздел «Правила»), в файле или через чат.

## Несущие принципы
- **Доказуемость.** Домысел не выдаётся за факт: гипотеза помечается, привязывается к источнику и перепроверяется по мере поступления фактов.
- **Повторяемость.** Готовый документ — детерминированная проекция зафиксированных фактов: одинаковые факты дают одинаковый результат.

## Критерий «проблемы»
- Симптом становится проблемой только при доказанном измеримом ущербе (обоснование бизнес-эффекта).

## Лестница статусов
- гипотеза → подтверждено цифровым следом → подтверждено интервью → согласовано заказчиком.

## Стандарт письма
- Раскрывать суть; полнота перечислений; экспертный язык без воды.

## Гейт синтеза
- Синтез не запускать, пока разведка не покрыла периметр.

## Политика правок
- Документы руками не правят — правят память и каркас, затем пересобирают.

## Язык и тон
- Русский, экспертный.
