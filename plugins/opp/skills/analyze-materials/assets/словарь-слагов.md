# Словарь слагов и правила чеканки кодов (итерация гейта Ф1)

Рабочий словарь (C9: операционализация; пополняется по мере доменов). Цель — чтобы два независимых
прогона чеканили ОДИНАКОВЫЕ коды: гейт Ф1 показал пересечение кодов 32% при смысловом 64%.

## Правила чеканки

1. Формат: `ПРЕФИКС-ДОМЕН[-УТОЧНЕНИЕ]`, латиница, слова через дефис, БЕЗ порядковых номеров.
2. Составные слова НЕ склеивать: `COST-MAP` (не `COSTMAP`), `KPI-FACT` (не `KPIFACT`).
3. Сокращение — из словаря ниже; нет в словаре — полное английское слово, и добавь кандидата в OQ
   для пополнения словаря.
4. Грануляция: «одно правило источника = одна строка» — код различай уточнением
   (`BR-SZ-VNEBUDGET`, `BR-SZ-DOPBUDGET`, `BR-SZ-DDS`), не сливай в общий `BR-SZ`.
5. Прежде чем чеканить — сверься с «уже_в_памяти»: существующий смысловой эквивалент = его код.

## Словарь доменных сокращений (финансовый контур 1С/ERP)

| Слаг | Значение | | Слаг | Значение |
|---|---|---|---|---|
| BDR | бюджет доходов и расходов | | MARGIN | маржа/доходность |
| BDDS | бюджет движения ДС | | COST | затраты/себестоимость |
| DDS | движение денежных средств | | PL | P&L (прибыли и убытки) |
| KPI | ключевые показатели | | FOT | фонд оплаты труда |
| UPZ | управление затратами (УПЗ) | | SZ | служебная записка |
| CFO | ЦФО | | VGO | внутригрупповые обороты |
| NSI | НСИ/справочники | | ZRDS | заявка на расход ДС |
| BUDGET | бюджет (общий) | | INVEST | инвестиции/CAPEX |
| DOPBUDGET | дополнительный бюджет | | TREASURY | казначейство |
| FCAST | прогноз | | LIMIT | лимиты |
| DEV | разработка/доработки | | RELEASE | релизный процесс |
| MASTER | мастер-система | | DUP | дубликаты |
