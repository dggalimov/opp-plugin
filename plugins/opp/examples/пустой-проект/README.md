# Пустой проект (пример рабочего пространства)

Это образец **рабочего пространства заказчика** в самом начале — без единого факта. Папка `memory/`
пуста.

Используется для самопроверки: контролёр целостности на пустой памяти **обязан** проходить без
замечаний (`./opp verify`). Это базовая гарантия — пустой проект всегда «зелёный».
