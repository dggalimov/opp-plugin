# OPP

Плагин Claude Code для обследования и ведения проекта внедрения 1С одним архитектором с ИИ:
второй мозг, анализатор входящей информации, технический писатель.

## Команды

- `/opp:new-project` — онбординг нового проекта.
- `/opp:analyze-materials` — анализ входящих материалов.
- `/opp:plan-meetings` — планирование цикла встреч.
- `/opp:analyze-meeting` — разбор проведённой встречи.
- `/opp:synthesis` — синтез фактов в требования и гипотезы.

Подробный гайд пользователя — `reference/guide/руководство.md`.

## Принципы

Доказуемость — в готовых документах только подтверждённые факты с указанием источника.
Повторяемость — одинаковые факты дают одинаковый результат.

Установка и требования — см. `УСТАНОВКА.md`.
